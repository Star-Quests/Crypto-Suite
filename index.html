ℹ️ How it works

• If you input units and current rate, balance = units × rate.
• To compute required pump: required increase per unit = desired_profit ÷ units (then % = increase ÷ current_rate ×100).
• If you input invested USD, units = invested ÷ current_rate.
All calculations use high-precision arithmetic to avoid rounding errors. Positive results are shown in green, negatives in red. Emojis help you quickly scan results. ✅
